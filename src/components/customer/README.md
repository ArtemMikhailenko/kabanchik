# Customer Dashboard Components

Создан набор модульных компонентов для страницы заказчика на основе дизайна Figma.

## Компоненты

### TabNavigation
Навигация по табам с четырьмя секциями:
- At work
- New Order  
- Selected Specialists
- Order History

### CreateOrderForm
Форма создания нового заказа с двумя состояниями:
- **Свёрнутая**: показывает краткое описание и кнопку "Create an order"
- **Развёрнутая**: полная форма с полями:
  - Выбор категории
  - Описание задачи
  - Детальное описание
  - Конфиденциальные данные
  - Загрузка файлов

### PopularOrderCard
Карточка популярного заказа с:
- Цветной иконкой (автоматически генерируется цвет)
- Названием заказа
- Местоположением
- Дедлайном
- Статусом
- Кнопкой "review"

### SpecialistCard
Карточка специалиста с:
- Аватаром
- Именем
- Местоположением
- Количеством отзывов
- Процентом положительных отзывов
- Рейтингом звёздами
- Кнопкой "job offer"

### Pagination
Компонент пагинации с:
- Кнопками навигации
- Номерами страниц
- Активной страницей
- Многоточием для большого количества страниц

## Цветовая схема

Соответствует дизайну Figma:
- `#282a35` - основной текст
- `#55c4c8` - кнопки и акценты
- `#ffa657` - оранжевые элементы
- `#a3a3a3` - вторичный текст
- `#f8f8f8` - фон страницы

## Использование

```tsx
import {
  TabNavigation,
  CreateOrderForm,
  PopularOrderCard,
  SpecialistCard,
  Pagination
} from '@/components/customer'
```

Все компоненты поддерживают TypeScript и имеют соответствующие интерфейсы для props.
